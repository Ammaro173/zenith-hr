# Web Interface Guidelines Audit

This skill checks for compliance with polished UI/UX rules (animations, accessibility, performance).

## How to Use

1.  **Read Rules**: The rules are inlined in `rules.md` in this directory.
2.  **Audit**: Check your target files against the rules.
3.  **Report**: List findings in `file:line` format.

## Rule Source
This skill uses a local copy of rules originally from `vercel-labs/web-interface-guidelines`.
