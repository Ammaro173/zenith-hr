# AGENTS.md

> Zenith HR is a manpower request and HR management system.

## Stack
- **Runtime**: Bun (Package Manager & Test Runner)
- **DB**: PostgreSQL + Drizzle ORM
- **API**: Elysia + oRPC
- **Frontend**: Next.js (App Router)
